## 2024-05-16 - Icon-Only Button Accessibility Pattern
**Learning:** In this codebase, the standard pattern for accessible icon-only interactive elements requires both an `aria-label` and specific Tailwind utility classes for focus states: `focus:outline-none focus-visible:ring-2 focus-visible:ring-accent focus-visible:ring-offset-1`.
**Action:** Ensure all newly created or existing icon-only buttons adhere to this pattern to guarantee keyboard navigation support and screen reader accessibility.
